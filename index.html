📌 Como usar

  1. Acesse o IMDb: Visite www.imdb.com
  2. Pesquise o filme: Use a barra de busca para encontrar o filme que deseja assistir
  3. Copie o ID: Na página do filme, copie o ID da URL. Por exemplo:
    https://www.imdb.com/title/tt0317219/...

    O ID é tt0317219

  4. Cole aqui: Insira o ID no campo acima e clique em "Carregar"
